# API REST Node / Express / TypeScript — Todo App

Mini projet backend visant à mettre en pratique le développement d’une API REST avec Node.js, Express et TypeScript, avec une évolution vers une architecture fullstack intégrant un frontend React.

Stack:
-Node.js
-Express
-TypeScript

Architecture backend:
-controllers/
-routes/
-datas/
-types/


Installation:
-npm install
-npm run dev
